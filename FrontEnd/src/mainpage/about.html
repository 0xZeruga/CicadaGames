About us

Cicada is a decentralized and anonymous organization that strives to develop games on the blockchain. Nowadays to many games on blockchain are simply games that aim to make the creators wealthy by featuring lottery element without providing a meaningful experience. We consider this to be fraud and to change this."

Installation

In order to participate in the network you need to have the Metamask browser addon installed. The addon can be found here."

Network

Cicada runs on the Gochain network and therefore allows for way faster transactions speeds than Ethereums Mainnet. The Gochain network also enable us to have much lower fees than those on Ethereums network. "

Core Principle

We aim to run as cheaply as possible but in order to further development of the platform there are ingame purchases that allows for extra functionality. We at Cicada are very strict with ensuring that these transactions are completely optional and do not grant a player advantage over anyone else."
